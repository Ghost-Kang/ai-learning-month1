# 最终总结报告：AI 学习第一个月

## 1. 项目背景

这个仓库记录了我第一个月在 Python 和数据相关方向上的学习过程。  
我把四周的练习和项目整理在一起，形成了一个阶段性 GitHub 作品集。

## 2. 学习路径

### 第一周：Python 基础
第一周主要完成了 20 个基础练习题，内容涵盖：

- 变量与数据类型
- 字符串格式化
- 条件判断与循环
- 列表与字典操作
- 文件读写
- CSV / JSON 处理
- 正则表达式
- 小工具脚本开发

### 第二周：数据采集与清洗
第二周围绕豆瓣 Top250 数据展开，主要内容包括：

- 网页请求与解析
- 电影信息提取
- 数据保存到 CSV / Excel
- 数据清洗与标准化
- 原始数据与处理后数据的整理

### 第三周：数据分析与可视化
第三周基于清洗后的数据进行了分析，主要内容包括：

- 评分分布分析
- 年代趋势分析
- 导演作品数统计
- 类型词云分析
- 评价人数与评分关系分析
- Notebook 报告展示

### 第四周：工具项目开发
第四周我完成了一个命令行数据清洗工具 `data-cleaner`，这个项目相比前几周更强调“工具化”和“模块化”设计，主要实现了：

- CSV 数据读取与清洗
- 去重与缺失值处理
- 图表输出
- HTML 报告生成
- 命令行参数支持
- 基础单元测试

这个项目让我进一步理解了如何把数据处理逻辑封装成可复用的工具，而不仅仅是一次性的分析脚本。
