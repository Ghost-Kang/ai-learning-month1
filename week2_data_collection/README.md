# 第二周：数据采集与清洗

本周主要围绕电影数据采集和清洗展开，目标是完成一个从网页获取数据到结构化保存的完整流程。

## 主要项目

### 豆瓣电影 Top250 数据采集器

一个基于 Python 的爬虫实战项目，用于抓取豆瓣电影 Top250 数据，并进行清洗、去重后保存为 CSV 文件

## 主要内容

- 发送网页请求
- 解析 HTML 页面
- 提取结构化字段
- 保存为 CSV
- 导出 Excel
- 对原始数据进行清洗与标准化


## 项目结构

```text
douban-scraper/
├── main.py             # 统一入口
├── scraper.py          # 爬虫主程序
├── cleaner.py          # 数据清洗脚本
├── utils.py            # 工具模块：日志配置、Session 创建
├── config.py           # 配置文件
├── requirements.txt    # 项目依赖
├── README.md
└── data/
    └── douban_top250.csv
```
